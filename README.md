# 👨‍💼 Employee Attrition Prediction using HR Analytics

## 📌 Project Overview

This project focuses on predicting employee attrition using Machine Learning algorithms. The goal is to identify employees who are likely to leave the organization based on various demographic, job-related, and performance factors.

This project demonstrates skills in Data Analysis, Exploratory Data Analysis (EDA), Data Preprocessing, Machine Learning Model Building, Evaluation, and Prediction.

---

## 🎯 Objectives

* Analyze employee data using Exploratory Data Analysis (EDA)
* Handle missing values and preprocess the dataset
* Perform feature encoding and transformation
* Train multiple Machine Learning models
* Compare model performance using evaluation metrics
* Predict employee attrition accurately
* Assist HR departments in making data-driven decisions

---

## 📊 Dataset Information

The dataset contains employee-related information such as:

* Age
* Gender
* Department
* Job Role
* Monthly Income
* Education
* Marital Status
* Job Satisfaction
* Work-Life Balance
* Years at Company
* Attrition Status (Target Variable)
